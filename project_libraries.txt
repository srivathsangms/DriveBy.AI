DriveBy.AI - Project Dependencies

============================================================
BACKEND (Python / FastAPI)
path: /backend/requirements.txt
============================================================
1. fastapi - Modern, fast (high-performance), web framework for building APIs with Python 3.7+ based on standard Python type hints.
2. uvicorn[standard] - ASGI web server implementation for Python.
3. gunicorn - Python WSGI HTTP Server for UNIX.
4. sqlalchemy - The Python SQL Toolkit and Object Relational Mapper.
5. alembic - A database migration tool for SQLAlchemy.
6. python-jose[cryptography] - A JavaScript Object Signing and Encryption implementation in Python.
7. passlib[bcrypt] - Password hashing library for Python.
8. python-dotenv - Reads key-value pairs from a .env file and can set them as environment variables.
9. pydantic-settings - Settings management using Pydantic.
10. psycopg2-binary - PostgreSQL database adapter for the Python programming language.

============================================================
AI ENGINE (Python / Computer Vision)
path: /driver_ai/requirements.txt
============================================================
1. opencv-python - Wrapper package for OpenCV python bindings.
2. mediapipe - Cross-platform, customizable ML solutions for live and streaming media (Face Mesh).
3. torch - Tensors and Dynamic neural networks in Python with strong GPU acceleration.
4. torchvision - Image and video datasets and models for torch deep learning.
5. numpy - The fundamental package for scientific computing with Python.
6. onnxruntime - High performance scoring engine for Open Neural Network Exchange (ONNX) models.
7. pygame - A set of Python modules designed for writing video games (Used for audio alerts).
8. requests - HTTP library for Python.
9. mouse - Hook into and control the mouse (if used for interaction features).
10. fer - Facial Expression Recognition library.
11. tensorflow - End-to-end open source platform for machine learning.
